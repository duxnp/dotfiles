# Back-up

1. Run: `backup.sh`
2. Save Contacts
    + Edit > Select All
    + File > Export... > Export vCard...
    + Copy to `~/Workspace/preferences/private`
3. `gitpush` dotfile repository changes
3. Copy to External HDD
    + `~/Workspace`
    + `~/Pictures`
    + `~/Movies`
    + `~/Music`
